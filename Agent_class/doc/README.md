## 技术栈总结

### 核心库
- **LangChain**：对话链、记忆管理、工具调用
- **Ollama**：本地LLM模型服务
- **LanceDB**：向量数据库存储
- **Express**：API服务器框架

### 关键技术
1. **流式输出**：实时响应显示
2. **向量检索**：相似度搜索和RAG
3. **工具调用**：Function Calling模式
4. **智能决策**：意图识别和路由
5. **模块化架构**：可维护的代码组织

### 工程实践
1. **错误处理**：分级错误处理和恢复
2. **性能优化**：缓存、连接池、超时控制
3. **配置管理**：分层配置系统
4. **监控日志**：结构化日志和性能监控
